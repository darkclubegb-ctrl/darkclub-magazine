# Code Visualizer
Analisa a estrutura do projeto e gera visualizações claras da arquitetura, fluxo de dados e hierarquia de componentes.

## Capacidades:
- Análise de grafo de dependências.
- Visualização de hierarquia de pastas.
- Mapeamento de fluxo de renderização de componentes.
- Identificação de gargalos arquiteturais.
