# SaaS Boilerplate Manager
Gerencia a infraestrutura base de um SaaS, incluindo autenticação, sistemas de pagamento e multi-tenancy.

## Capacidades:
- Implementação de NextAuth/Clerk.
- Integração com Stripe/MercadoPago.
- Configuração de subscrições e planos.
- Gestão de organizações e usuários.
