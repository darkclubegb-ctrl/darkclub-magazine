# Git Commander
Especialista em fluxos Git, gerenciamento de branches, resolução de conflitos e automação de Pull Requests.

## Capacidades:
- Inicialização e configuração de repositórios.
- Estratégias de branching (Git Flow / Trunk Based).
- Squash, Rebase e Gerenciamento de Histórico.
- Sincronização com repositórios remotos.
