# Figma Bridge
Conecta o design ao desenvolvimento, extraindo tokens, cores, tipografias e layouts diretamente de arquivos Figma.

## Capacidades:
- Identificação de tokens de design.
- Exportação automatizada de assets.
- Conversão de layouts Figma para JSX/Tailwind.
- Sincronização de design system.
