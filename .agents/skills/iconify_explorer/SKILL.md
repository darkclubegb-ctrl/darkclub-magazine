# Iconify Explorer
Facilita a busca, seleção e implementação de ícones otimizados de múltiplas bibliotecas via Iconify.

## Capacidades:
- Integração com @iconify/react ou similares.
- Seleção de ícones por contexto.
- Otimização de pacotes de ícones.
- Padronização visual de iconografia.
