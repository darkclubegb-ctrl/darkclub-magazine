# Web Browser
Capacita o agente a navegar na web, extrair informações de sites e testar interfaces de usuário em tempo real.

## Capacidades:
- Navegação autônoma em múltiplos domínios.
- Extração de dados (Web Scraping).
- Captura de tela para validação visual.
- Interação com elementos da DOM.
